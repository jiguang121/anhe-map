# 天才俱乐部 Genius Club

这是一个静态 H5 V2，用于验证“隐私 / 心事 / 爆料”的匿名内容浏览、评论、会员墙和后台内容管理体验。

## 已实现功能

- 首页三个漂浮入口：隐私、心事、爆料
- 分类内容流
- 匿名发布内容
- 查看详情
- 匿名评论
- 游客浏览指定条数后触发会员墙
- 演示登录 / 演示会员
- 独立后台页：admin.html
- 后台可管理首页文案、入口名称、入口小字、文章、评论、会员墙文案、免费浏览条数
- 支持导出 / 导入 JSON 内容数据
- 数据保存在浏览器 localStorage

## 页面入口

前台：

```txt
/index.html
```

后台：

```txt
/admin.html
```

管理密码：

```txt
genius-admin
```

## 当前版本的重要限制

当前版本是“静态站 + 本地后台”。后台修改内容后，会保存在当前浏览器的 localStorage 里。

也就是说：

- 你自己这台电脑能看到修改结果
- 其他用户不会自动看到你本机后台改的内容
- 清除浏览器缓存后，本机修改可能丢失

如果要做到“你在后台一改，所有访问者都看到最新内容”，下一步必须接真实数据库和后端鉴权。

## 正式上线下一步

1. 先用 GitHub Pages / Netlify / Vercel 部署静态站。
2. 接数据库：Supabase、LeanCloud、腾讯云开发或自建后端。
3. 后台登录改为真实账号权限，不再把密码写在前端。
4. 文章、评论、会员状态、付费状态全部进入数据库。
5. 接微信 / 支付宝 / 手机号登录与真实支付。
