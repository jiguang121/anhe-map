# 天才俱乐部 Genius Club

这是一个静态 H5 MVP，用于验证“隐私 / 心事 / 爆料”的匿名内容浏览与互动体验。

## 已实现功能

- 首页三个漂浮入口：隐私、心事、爆料
- 分类内容流
- 匿名发布内容
- 查看详情
- 匿名评论
- 游客浏览 5 条后触发会员墙
- 演示登录 / 演示会员
- 本地演示后台，可手动增加内容
- 数据保存在浏览器 localStorage

## 演示后台

后台入口在首页底部。

管理密码：

```txt
genius-admin
```

注意：当前后台只是前端演示，密码写在前端代码里，不适合正式上线。正式版本需要后端数据库和登录鉴权。
